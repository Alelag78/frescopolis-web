# Frescópolis - Supermercado Online

## Descripción
Frescópolis es un proyecto de página web para un supermercado local que muestra productos frescos, bebidas y artículos de limpieza.  
El diseño se realizó primero como mockup en Figma y luego se implementó usando **HTML, CSS y JavaScript**.  
Incluye animaciones, interactividad para promociones y un diseño responsive adaptable a móviles y tablets.

---

## Funcionalidades
- **Página de inicio** con productos destacados y banner de promociones animado.
- **Categorías de productos** con scroll horizontal responsive.
- **Detalle de producto** con información adicional y aplicación de descuentos mediante JavaScript.
- **Navegación clara y fácil de usar**, con cabecero fijo y enlaces a secciones principales.
- Diseño **responsive** usando CSS Grid, Flexbox y media queries.
